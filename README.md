# attendance-management-system

A modern QR Code-based Attendance Management System built with:

- Next.js 16 (PWA)
- .NET 10 Minimal API
- Clean Architecture
- PostgreSQL
- Docker
- .NET Aspire

## Features

- Authentication
- Student Management
- Lecturer Management
- QR Code Attendance
- Email Verification
- Attendance Reports
